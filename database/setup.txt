Test Local MongoDB Database Setup Tutorial

1. get mongodb and robomongo (you'll love it, like you love me)
2. start mongod
3. start mongo
4. type these commands:

	use TESTtf2stadium
	db.createUser({ user: "tf2stadium", pwd: "dickbutt", roles: [ { role: "readWrite", db: "TESTtf2stadium" } ]})

	use tf2stadium
	db.createUser({ user: "tf2stadium", pwd: "dickbutt", roles: [ { role: "readWrite", db: "tf2stadium" } ]})

5. test db_test
6. if no errors, scream hallelujah
